# Smoker Survey Form

A ReactJS survey form for smokers built using `react-hook-form` and `yup` for validation.

## Features

- Form state management with `react-hook-form`
- Schema-based validation with `yup`
- Responsive survey layout

## Getting Started

### 1. Install dependencies

```bash
npm install react-hook-form yup @hookform/resolvers
```

### 2. Run the app

```bash
npm start
```

## Tech Stack

- ReactJS
- React Hook Form
- Yup
